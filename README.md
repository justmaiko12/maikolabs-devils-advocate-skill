# MaikoLabs Devils Advocate Skill

A pressure-test skill for challenging weak assumptions before you build,
publish, automate, or commit to a direction.

Use this when you want the AI to push back productively instead of just agreeing.

## Install

```bash
mkdir -p ~/.codex/skills
git clone https://github.com/justmaiko12/maikolabs-devils-advocate-skill ~/.codex/skills/maikolabs-devils-advocate-skill
```

Then start a new Codex chat and ask it to use the Devils Advocate skill.

## What It Helps With

- Finding blind spots
- Catching unclear goals
- Reducing over-scoping
- Testing whether an idea serves the audience
- Turning vague excitement into a sharper plan
